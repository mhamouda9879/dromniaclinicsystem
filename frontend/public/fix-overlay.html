Quick Fix:

Run these commands in your browser console (F12):

// Remove any blocking overlays
document.querySelectorAll('*').forEach(el => {
  const style = window.getComputedStyle(el);
  if (style.position === 'fixed' && (style.zIndex > 1000 || style.background !== 'rgba(0, 0, 0, 0)')) {
    console.log('Found potential overlay:', el);
  }
});

// Force enable pointer events
document.body.style.pointerEvents = 'auto';
document.body.style.userSelect = 'auto';
document.querySelector('#root').style.pointerEvents = 'auto';

// Remove any invisible overlays
const allElements = document.querySelectorAll('*');
allElements.forEach(el => {
  if (el.style.pointerEvents === 'none') {
    el.style.pointerEvents = 'auto';
  }
});

console.log('Fixed! Try clicking now.');

Manual Steps:

  1. Press F12 or Ctrl+Shift+I (Windows) / Cmd+Option+I (Mac)
  2. If DevTools doesn't open, try Ctrl+Shift+J (Windows) / Cmd+Option+J (Mac)
  3. Go to Console tab
  4. Paste the code above and press Enter
  5. Try clicking again
